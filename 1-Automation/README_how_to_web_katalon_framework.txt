how to install katalon web automation framework (free):

1. install katalon studio attached or on the web:
https://www.katalon.com/web-testing/

2. open katalon studio and check for updates

3. import test project / test case:

4. install browser addons, e.g.
https://chrome.google.com/webstore/detail/katalon-recorder-selenium/ljdobmomdgdljniojadhoplhkpialdid
https://addons.mozilla.org/en-GB/firefox/addon/katalon-automation-record/?src=search

5. run test by selecting browser(s):
pre-requisites:
recheck login credentials/password encryption
recaptcha step has to be done manually (test has a 120seconds timeout to do this step)

5. reporting can be checked in the tool or online:
https://analytics.katalon.com/welcome

Notes:
1.problems with recaptcha, this version is really slow and annoying for the user/automation, besides cannot use checkbox visible/verification
consider using:
https://www.google.com/recaptcha/intro/v3.html

2.Test case can get optimized in terms of run time (wait vs visible elements)

3. Also unexpected problems with credentials
